CLI

Commands

A compact overview of the main Sentinel CLI commands before the full reference is expanded.

Main ingestion commands

CommandPurpose
sentinel dockerAuto-discover Docker containers and triage their logs
sentinel stdinRead logs from piped stdin
sentinel run -- <command>Run a child process and capture both stdout and stderr
sentinel watch <glob...>Tail log files continuously

Docker inspection commands

CommandPurpose
sentinel docker listShow watched and ignored containers
sentinel docker explain [container...]Explain why a container is or is not being watched

Inspection and maintenance

CommandPurpose
sentinel uiStart the local web UI
sentinel historyList recent fingerprints/issues
sentinel explainInspect an issue in more detail
sentinel checkCheck runtime configuration and environment
sentinel updateUpdate the installed binary from published releases

Scope note
